2. Actors / Users
#	Actor / User Type	Role / Main Functions	Example Use Case
1	ลูกค้า (Customer / Member)	เลือกสินค้า, สั่งซื้อ, ดูโปรโมชั่น, ใช้แต้มสมาชิก, ชำระเงิน, ติดตามสถานะคำสั่งซื้อ	ค้นหาสินค้า, ใช้แต้ม ALL Member, ชำระเงินผ่าน TrueMoney
2	พนักงานร้าน (Store Staff)	รับคำสั่งซื้อ, เตรียมสินค้า, เช็คสต็อก, จัดรายการสั่งซื้อ	รับออเดอร์, อัปเดตสถานะ “กำลังจัดเตรียม”
3	ไรเดอร์ / ผู้ส่งสินค้า (Delivery Rider)	รับออเดอร์, นำส่งสินค้า, อัปเดตสถานะการส่ง	กด “รับออเดอร์”, ส่งของให้ลูกค้า, อัปเดต “จัดส่งสำเร็จ”
4	แอดมิน (Admin / Back Office)	ตรวจสอบการทำงานร้านค้า, ระบบสต็อก, โปรโมชั่น, ตรวจสอบปัญหา	อนุมัติร้าน, จัดการโปรโมชั่น, ตรวจสอบปัญหาออเดอร์
5	Customer Service	ช่วยเหลือลูกค้าปัญหาออเดอร์, การชำระเงิน, สินค้าขาด	เปิดดูออเดอร์, ประสานงานร้าน/ไรเดอร์, คืนเงิน
3. External Systems / Services
#	External System / Service	Function	Example Interaction
1	Payment Gateway	ประมวลผลการชำระเงินทุกประเภท	แอปส่งคำขอ → ตอบกลับ “ชำระสำเร็จ/ไม่สำเร็จ”
2	TrueMoney Wallet	ชำระเงิน เติมเงิน เชื่อมยอดเงิน	ตรวจสอบยอดเงิน → ตอบว่า “พอจ่าย/ยอดไม่พอ”
3	All Member	ตรวจสอบแต้มสะสม, คูปอง, สิทธิ์สมาชิก	ตรวจแต้ม → ตอบว่า “มี 150 แต้ม, 2 คูปอง”
4	Inventory / ERP	ตรวจสอบสต็อกสินค้า, ราคา, โปรโมชั่น	ถามสินค้ารหัส 1 → ตอบ “เหลือ 12 ชิ้น”
5	Promotion Engine	คำนวณส่วนลด, โปรโมชั่น	ส่งรายการสินค้า → ตอบว่า “โปร ซื้อ 2 เหลือ 25 บาท”
6	Google Maps API	หาร้านใกล้เคียง, คำนวณระยะทาง	ส่งตำแหน่ง → ตอบ “ร้านใกล้สุด 300 เมตร”
7	7Delivery	จัดการไรเดอร์, เส้นทางจัดส่ง, เวลาโดยประมาณ	ส่งที่อยู่ลูกค้า → ตอบ “ไรเดอร์รับงานแล้ว”
8	SMS Gateway (OTP)	ส่งรหัส OTP ยืนยันตัวตน	แอปขอส่ง OTP → ส่ง OTP ไปมือถือ
9	FCM (Push Notification)	แจ้งเตือนโปรโมชั่นหรือสถานะออเดอร์	ส่งข้อความโปรโมชัน → แจ้งเตือนไปมือถือ
10	Social Login (Google/LINE/Apple)	ล็อกอินด้วยบัญชีโซเชียล	แอปขออนุญาตล็อกอิน → ยืนยันตัวตนสำเร็จ
11	CRM	บริการลูกค้า, เก็บประวัติการติดต่อ	ส่งเรื่องร้องเรียน → ระบบสร้างเคส “กำลังตรวจสอบ”

3. External Systems / Services
ลำดับ	ชื่อ External System / Service	ทำหน้าที่	ตัวอย่างการโต้ตอบกับระบบของเรา
1	ระบบชำระเงิน (Payment Gateway)	ประมวลผลการชำระเงินทุกรูปแบบ เช่น เงินสด, บัตรเครดิต, TrueMoney	แอปส่งคำขอให้ระบบตัดเงิน → ระบบตอบกลับว่า “ชำระสำเร็จ” หรือ “ชำระไม่สำเร็จ”
2	ระบบทรูมันนี่วอลเล็ท (TrueMoney Wallet)	ชำระเงินผ่านวอลเล็ท, เติมเงิน, เชื่อมยอดเงิน	แอปถามว่ามีเงินพอไหม → TrueMoney ตอบว่า “พอจ่าย” หรือ “ยอดเงินไม่พอ”
3	ระบบสมาชิก (All Member)	เก็บและตรวจสอบแต้มสะสม, คูปอง และสิทธิ์สมาชิก	แอปถาม “ตอนนี้ลูกค้ามีกี่แต้ม” → ระบบตอบ “มี 150 แต้ม และมีคูปอง 2 ใบ”
4	ระบบสต๊อกสินค้า (Inventory/ERP)	ตรวจสต็อกสินค้า, ราคาสินค้า, โปรโมชั่น	แอปถาม “สินค้ารหัส 1เหลือกี่ชิ้น?” → ระบบตอบ “เหลือ 12 ชิ้นที่สาขาใกล้คุณ”

8	ระบบ OTP (SMS Gateway)	ส่งรหัส OTP เพื่อยืนยันตัวตน	แอปขอส่ง OTP → ระบบส่ง OTP ไปที่มือถือของลูกค้า
9	ระบบแจ้งเตือน (FCM)	ส่ง Push Notification โปรโมชัน หรือแจ้งเตือนคำสั่งซื้อ	แอปส่งข้อความโปรโมชัน → ระบบแจ้งเตือนไปที่มือถือทันที
10	ระบบล็อกอินภายนอก (Google/LINE/Apple)	ใช้ล็อกอินด้วยบัญชีโซเชียล	แอปขออนุญาตล็อกอิน → ระบบตอบกลับว่า “ยืนยันตัวตนสำเร็จ”
11	ระบบบริการลูกค้า (CRM)	ให้บริการลูกค้า, เก็บประวัติการติดต่อ	แอปส่งเรื่องร้องเรียน → ระบบสร้างเคสและแจ้งว่า “กำลังตรวจสอบ”
